# GERMAN_LAB_FULL

## Проект Олега Германа

Этот каталог представляет собой полный проект измерительной лаборатории, созданный для платформы MOKO SE по автоматизации измерений.

### Содержимое

*   **German's Measurement** и **German's Measurement Full**: Директории с результатами измерений (`.mpr`), сгруппированными по датам.
*   **data**: Содержит данные, включая `Results.csv` и схему измерительной установки (`Схема измерения.png`).
*   **Python-скрипты**: Набор скриптов, автоматизирующих процесс измерений:
    *   `1 Настройка параметров.py`: Настройка параметров измерений.
    *   `2 Инициализация приборов.py`: Инициализация измерительного оборудования.
    *   `3 Измерения.py`: Выполнение основной последовательности измерений.
    *   `4 Создание отчета.py`: Генерация отчета по результатам измерений.
*   **Библиотечные файлы**: Модули `MFRT.py`, `MGPH.py`, `MOKO.py`, `MOSC.py`, предоставляющие вспомогательные функции для проекта.
*   **German's Measurement Full.msep**: Файл проекта MOKO SE.
*   **Отчет.docx**: Шаблон или пример отчета в формате Word.

### Назначение

Проект `GERMAN_LAB_FULL` является комплексным решением для проведения автоматизированных измерений и генерации отчетов на платформе MOKO SE. Суффикс "FULL" указывает на то, что это полная версия измерительного проекта.
